Why we use the Normalize CSS Script

The Normalize CSS Script resets any out of the ordinary styles, and it uses the browser's default styles. It helps the usability of the code because default variables do not have to be redefined. It also is used across many browsers.

What is Semantic Markup

Semantic markup is important to help the coder and others who read the code to understand what the code is for. The browser can recognize the markup to better understand the code. It reinforces the meaning of the code, so it is easily accessibile when looking at the HTML.
