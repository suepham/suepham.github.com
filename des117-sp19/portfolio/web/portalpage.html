Portal Page

Challenge Statement

Build a page that would include the links to all of the projects done in the class (major projects, exercises, links).

Solution Statement

I coded the page in HTML in the beginning because I was learning how to code with HTML. I continued to add links of my exercises throughout the quarter, so the page will have content to style while being a place I can refer back to when I code in the future. I created the header with a blue, purple, and green color scheme, so the page is visually interesting. I made the headers of the three categories have an overlay of the words, so that it stood out creating a visual hierarchy. I used a grid to make the categories because it evenly divided it on the page. I made the footer blue, so that it would fit the color scheme.
