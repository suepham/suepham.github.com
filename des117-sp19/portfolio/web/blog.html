Blog Website

Challenge Statement

Design and build a blog site that will be web and mobile friendly using only HTML and CSS.

Solution Statement

I first used Sketch to create the design for the website, so I can see what I am trying to work towards in HTML and CSS. I added a navigation that would separate the blog posts into categories, so it's easier for users to read the content they want. I used a grid system to layout the posts because it creates a clean and organized space for the blog. I added a footer with additional links that do not fit in the main navigation because users can find out more about me when they explore the site. I first marked up the blog with HTML because I needed to add in all of the content first, and I then started styling with CSS. I used one CSS sheet to design the front page and the article pages, so it would be more organized. I used a grid in CSS because it organized it the same way in my original design. I used fontawesome to add in the icons at the bottom of the page because the icons were more visually appealing for the user to view and click on.
