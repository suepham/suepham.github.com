Typography Poster

Challenge Statement

The project was to create a poster for the font “Weiss” that would communicate the essential characteristics of the typeface. Only type could be the graphical element in the poster, and the poster should reflect the nature of the typeface as intended by the designer.

Solution Statement

I placed the typeface name at the top because I wanted it to be the first visual element that the audience sees. I then placed Emil Rudolf underneath followed by the year it was published because the designer was more significant about the typeface. I used red in the alphabet to showcase the unique characters in the typeface, and the red letters connected with the designers’ name because it is made up of letters. Similarly, I also did this for the numbers and using yellow, so it would relate back to the year.
