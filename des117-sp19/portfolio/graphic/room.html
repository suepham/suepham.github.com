Interior of a Room

Challenge Statement

The project was to design a two-point perspective of an interior of a room. It had to use Adobe Illustrator, so I could learn how to use the software.

Solution Statement

I first sketched the interior of a room, so I could see what I had to do on Illustrator. I, then, used Illustrator to draw the outlines of the furniture, so I could make sure the design was correct before I added color. We had to make four versions of the room with different color schemes, so we could show our understanding of how colors interacted. I used purple and blue for analogous colors, blue and orange for complementary colors, and greyscale for monochromatic colors.
